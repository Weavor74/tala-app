## 3. Data Processing Rules

### 3.1. Sovereign-Only Memory
Memory facts (Mem0 / RAG) are stored **only on the local machine**. Vector embeddings are generated locally via Ollama (e.g., `nomic-embed-text`) to ensure that semantic representations of user data never leave the host.

### 3.2. Redaction at the Edge
The **Quantum Firewall** (`log_redact.ts`) is invoked at the moment of log generation. Any object containing keys such as `apiKey`, `token`, or `password` is recursively processed to replace values with `***`, preventing accidental persistence of credentials.

### 3.3. Retention and Deletion
Users maintain full sovereignty over their data. Deleting a chat session in the UI performs a recursive `fs.unlinkSync` on the corresponding JSON file and removes associated Goal Graph nodes.
